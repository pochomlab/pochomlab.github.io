# 🤝 PochomLab Collaborators Guide

PochomLabは、創作・イベント・記録をゆるやかに育てていくための場所です。

GitHubを使って、イベント記事やLog記事を
追加・更新することができます。

---

## 🌱 コラボレーターとしてできること

コラボレーターとして参加すると、以下のような記録を投稿できます。

- 🎫 イベント記事の投稿（告知・レポート）
- 🧪 Log記事の投稿（制作記録・活動ログ）
- 🌿 創作・展示・ZINE活動の共有

小さな活動でも問題ありません。
完成された成果だけでなく、過程や試行も大切な記録として扱っています。

---

## 🚀 参加方法

### 1. GitHubアカウントを用意する

参加には GitHub アカウントが必要です。

👉 https://github.com/

### 2. 管理者へ連絡する

参加をご希望の方は、管理者までご連絡ください。

- お問い合わせフォーム
- メール など

### 3. 招待を受ける

リポジトリへ招待されたら、コラボレーターとして参加できます。

---

## 📝 投稿の流れ（共通）

1. このリポジトリを開く
2. 新しいブランチを作成（またはFork）
3. 新しい記事ファイルを作成
4. テンプレートを元に内容を書く
5. Pull Request を作成
6. 内容確認後、公開されます

GitHubの操作に不安がある場合は、そのままご相談ください。

---

## 🎫 イベント記事の追加方法

イベント情報や出展記録を追加するときは、イベント記事テンプレートを使用します。

### テンプレート

[👉 イベント記事テンプレートを見る](https://pochomlab.com/ja/log/2026-event-registration-sample/ "イベント登録サンプル")

### 記載内容の例

- イベント名
- 開催日
- 会場
- 概要
- 関連リンク

---

## 🧪 Log記事の投稿

制作過程や活動記録を残すときは、Log記事テンプレートを使用します。

### テンプレート

[👉 Log記事テンプレートを見る](https://pochomlab.com/ja/log/2026-log-registration-sample "ログ登録サンプル")


### 記載内容の例

- 制作の進捗
- 試行錯誤
- 気づき
- 技術メモ

---

## 📁 保存先

- 🎫Event articles: `/src/content/events/`
- 🧪Log articles (JA): `/src/content/log-ja/`
- 🧪Log articles (EN): `/src/content/log-en/`

---

## 🔁 Pull Request の作成

1. ファイルを追加・編集
2. 「Create Pull Request」をクリック
3. タイトルと説明を記入
4. 送信

内容を確認後、問題なければ反映されます。

---

## 🆘 困ったとき

操作がわからない場合や不安な場合は、
お気軽にご相談ください。

GitHubに慣れていない方も歓迎しています。

---

## 🌿 Notes

PochomLabは、完成された作品だけでなく、
過程や試行も含めて残していくための場です。

小さな記録も、大切な灯として扱っています。

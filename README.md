# 🤝 PochomLab Collaborators Guide

PochomLab is a place for gently nurturing creative work, events, and documentation.

Using GitHub, collaborators can add and update event articles and log articles.

---

## 🌱 What You Can Do as a Collaborator

As a collaborator, you can contribute records such as:

- 🎫 Event articles (announcements and reports)
- 🧪 Log articles (creation records and activity logs)
- 🌿 Sharing creative work, exhibitions, and ZINE activities

Small activities are welcome too.  
PochomLab values not only finished results, but also the process and experimentation behind them.

---

## 🚀 How to Join

### 1. Prepare a GitHub account

A GitHub account is required to participate.

👉 https://github.com/

### 2. Contact the administrator

If you would like to join, please contact the PochomLab administrator.

- Contact form
- Email, etc.

### 3. Receive an invitation

Once you are invited to the repository, you can join as a collaborator.

---

## 📝 Posting Flow

1. Open this repository
2. Create a new branch (or fork the repository)
3. Create a new article file
4. Write your content using the template
5. Create a Pull Request
6. After review, your article will be published

If you are not comfortable with GitHub operations, please feel free to ask.

---

## 📁 File Locations

- 🎫 Event articles: `/src/content/events/`

- 🧪 Log articles (EN): `/src/content/log-en/`

---

## 🔁 Creating a Pull Request

1. Add or edit a file
2. Click “Create Pull Request”
3. Enter a title and description
4. Submit it

After review, it will be merged if everything looks good.

---

## 🆘 Need Help?

If you are unsure about anything or do not know how to proceed, please feel free to contact us.

People who are new to GitHub are also welcome.
